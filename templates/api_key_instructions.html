Google API Võtme Seadistamine

Tasuta Kasutamine: Google pakub igakuiselt $200 krediiti Maps API kasutamiseks, millest piisab umbes 40 000 marsruudi konverteerimiseks kuus ilma lisakuluta.

Selleks, et Google Maps GPX konverter järgiks korrektselt teid (mitte ei ühendaks teekonnapunkte sirgjoonega), peate seadistama Google API võtme juurdepääsuga suunajuhiste (Directions) API-le.

Samm 1: Looge Google Cloud Platform konto

  1. Minge Google Cloud Platform lehele
  2. Logige sisse oma Google kontoga või looge uus
  3. Looge uus projekt (või kasutage olemasolevat)

Samm 2: Aktiveerige Directions API

  1. Google Cloud Console'is navigeerige "APIs & Services" > "Library"
  2. Otsige "Directions API"
  3. Klikkige tulemuste seas "Directions API" peale
  4. Klikkige "Enable"

Samm 3: Looge API Võti

  1. Navigeerige "APIs & Services" > "Credentials"
  2. Klikkige "Create Credentials" > "API key"
  3. Luuakse uus API võti. Kopeerige see võti.

Samm 4: (Valikuline, kuid Soovitatav) Piirake Oma API Võtit

Turvalisuse huvides on hea tava piirata oma API võtit:

  1. Credentials lehel leidke oma API võti ja klikkige "Edit"
  2. "Application restrictions" all valige "IP addresses" ja lisage oma IP-aadress
  3. "API restrictions" all valige "Restrict key" ja valige rippmenüüst "Directions API"
  4. Klikkige "Save"

Samm 5: Lisage API Võti Rakendusele

API võtme lisamiseks rakendusse on kaks võimalust:

Variant 1: Keskkonnamuutuja

Määrake GOOGLE_MAPS_API_KEY keskkonnamuutuja enne rakenduse käivitamist:

Windows:

set GOOGLE_MAPS_API_KEY=your_api_key_here

macOS/Linux:

export GOOGLE_MAPS_API_KEY=your_api_key_here

Variant 2: .env Faili Loomine

  1. Looge fail nimega .env rakenduse kataloogis
  2. Lisage järgmine rida faili: 
    GOOGLE_MAPS_API_KEY=your_api_key_here
  3. Salvestage fail

Samm 6: Taaskäivitage Rakendus

Pärast API võtme seadistamist taaskäivitage rakendus muudatuste rakendamiseks.

Tasuta Kasutamise Piirangud

  • Google pakub $200 igakuist krediiti Google Maps Platform API-de kasutamiseks
  • Directions API maksab $5 iga 1000 päringu kohta
  • Tasuta krediidiga saate teha ligikaudu 40 000 marsruudi konverteerimist kuus ilma kuluta
  • Enamiku isikliku kasutuse jaoks peaks sellest piisama
Märkus: Kui ületate tasuta piirangud, peate oma Google Cloud kontol aktiveerima arvelduse. Kui olete kulude pärast mures, jälgige oma kasutust.

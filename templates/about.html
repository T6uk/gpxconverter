Google Maps GPX Konverterist

Lihtne tööriist matkajatele ja seiklejatele

See tööriist võimaldab teil konverteerida Google Maps marsruudid GPX-failideks, mida saab kasutada erinevate GPS-seadmete, aktiivsusmonitoride ja navigatsioonirakenduste puhul. See on eriti kasulik marsruutide planeerimiseks matkamiseks, jooksmiseks, jalgrattasõiduks või muuks tegevuseks, mis nõuab navigeerimist.

Kuidas See Töötab

Konverter eraldab teekonnapunktid Google Maps URL-idest ja teisendab need GPX (GPS Exchange Format) formaati. GPX on levinud standard GPS-andmete vahetamiseks rakenduste ja seadmete vahel.

Kui kleebite Google Maps URL-i, teeb rakendus järgmist:

  1. Eraldab URL-ist kõik teekonnapunktid
  2. Tuvastab liikumisviisi (kõndimine, jalgrattasõit, autosõit)
  3. Loob GPX-faili õigete rajalõikude ja metaandmetega
  4. Võimaldab teil faili alla laadida ja seadmes kasutada

Oluline Märkus

Selleks, et konverter järgiks teid (mitte ei tõmbaks sirgjoonelisi ühendusi punktide vahel), peate seadistama Google API võtme. See on vajalik ühilduvuseks GPS-seadmetega nagu Garmin.

Konfigureerige oma API võti konverteri lehe täpsemate seadete jaotises.

Näpunäited Garmin Connecti Jaoks

Kui kasutate Garmin Connecti:

  1. Garmin Connectis minge "Treening" → "Kursused" → "Impordi"
  2. Valige sellest konverterist alla laaditud GPX-fail
  3. Salvestage see kursusena (mitte tegevusena)
  4. Sünkroonige oma Garmini seade
  5. Oma seadmes leidke kursus navigatsiooni või treeningu valikute alt

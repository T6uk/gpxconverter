Instructions:

  1. Go to Google Maps
  2. Create your route by setting start and end points, and add waypoints if needed
  3. Copy the URL from your browser's address bar
  4. Paste the URL in the field above and click "Convert to GPX"
  5. Save the downloaded GPX file to use on your device

Tips for better results:

  • For walking, cycling or driving routes, make sure to select that mode in Google Maps
  • Add waypoints to make your route follow a specific path
  • The converter handles complex routes with multiple waypoints
  • For Garmin Connect: Import the GPX file as a "Course" rather than an "Activity"

About Google Maps to GPX Converter

A simple tool for outdoor enthusiasts and adventurers

This tool allows you to convert Google Maps routes into GPX files that can be used with various GPS devices, fitness trackers, and navigation apps. It's especially useful for planning routes for hiking, running, cycling, or any other outdoor activity that requires navigation.

How It Works

The converter extracts waypoints from Google Maps URLs and converts them into the GPX (GPS Exchange Format) format. GPX is a common standard for exchanging GPS data between applications and devices.

When you paste a Google Maps URL, the application:

  1. Extracts all waypoints from the URL
  2. Detects the travel mode (walking, cycling, driving)
  3. Creates a GPX file with proper track segments and metadata
  4. Delivers the file for you to download and use on your device

Road-Following Routes

Important Note

For the converter to follow roads (instead of straight lines between points), you need to set up a Google API key. This is necessary for compatibility with GPS devices like Garmin.

Configure your API key in the advanced options section on the converter page.

Tips for Garmin Connect

If you're using Garmin Connect:

  1. In Garmin Connect, go to "Training" → "Courses" → "Import"
  2. Select the GPX file downloaded from this converter
  3. Save it as a course (not an activity)
  4. Sync your Garmin device
  5. On your device, find the course under navigation or training options
